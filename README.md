# Modern Gallery

A simple and modern gallery website built with React and Tailwind CSS.

## Features

- Animated landing page
- Dynamic image gallery using Picsum API
- Contact form with EmailJS
- About page
- Custom 404 page
- Responsive dark UI

## Tech Stack

- React
- Tailwind CSS
- React Router DOM
- Axios
- EmailJS

## Run Project

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file and add:

```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```
